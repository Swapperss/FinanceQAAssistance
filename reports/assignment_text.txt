 Practical Fine-Tuning Assignment 
 Build a Domain-Specific AI Assistant using Unsloth 
 Objective 
 In this assignment, you need to build a practical domain-specific AI assistant by fine-tuning an 
 open-source Large Language Model using  Unsloth  . 
 The goal is to understand a real industry-style fine-tuning workflow: 
 ●  Raw domain data preparation 
 ●  Non-instruction fine-tuning 
 ●  Instruction dataset preparation 
 ●  LoRA /DoRA/ QLoRA fine-tuning 
 ●  Preference dataset creation 
 ●  DPO/ORPO alignment 
 ●  Model evaluation 
 ●  Before-vs-after comparison 
 ●  Saving and using the fine-tuned model 
 Business Problem 
 You are working as a  GenAI Engineer  . 
 Your company wants to build an internal AI assistant that can answer questions for one specific 
 domain. 
 The assistant should understand domain-specific terminology, answer user questions clearly, 
 and give better responses than the base model. 
 Choose Any One Domain

 Step 4: Create Instruction Dataset 
 Now create an instruction-response dataset for your selected domain. 
 This dataset teaches the model how to answer user questions. 
 Minimum Requirement 
 Create at least: 
 ●  100 instruction-response examples 
 Each example should contain: 
 ●  instruction / question 
 ●  response / answer 
 Example Format 
 { 
 "instruction": "How can I apply for sick leave?", 
 "response": "You can apply for sick leave through the HR portal by selecting the sick leave 
 option and submitting the required details." 
 } 
 You can create the dataset manually, from public documents, from public datasets, or with the 
 help of AI. 
 But you must clean and verify the dataset before using it.

 Step 8: Create Preference Dataset for DPO 
 Now create a preference dataset for alignment. 
 This dataset teaches the model which answer is better and which answer is weaker. 
 Minimum Requirement 
 Create at least: 
 ●  50 preference examples 
 Each example should contain: 

===== PAGE 9 =====
 ●  prompt 
 ●  chosen response 
 ●  rejected response 
 Example Format 
 { 
 "prompt": "How can I cancel my order?", 
 "chosen": "You can cancel your order from the Orders section before it is shipped. If the order 
 has already been shipped, you can request a return after delivery.", 
 "rejected": "Just ignore the order." 
 } 
 Guidelines 
 The chosen response should be: 
 ●  Correct 
 ●  Helpful 
 ●  Safe 
 ●  Professional 
 ●  Domain-specific 
 The rejected response can be: 
 ●  Wrong 
 ●  Incomplete 
 ●  Unsafe 
 ●  Rude 
 ●  Too generic 
 ●  Not domain-specific 
 Deliverable 
 data/preference_dataset.jsonl
